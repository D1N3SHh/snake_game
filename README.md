# SSSNAKE GAME

Simple snake game made using [pygame](https://github.com/pygame/pygame)

## Installation

Use [pip](https://pip.pypa.io/en/stable/) to install all required packages.

```bash
pip install -r requirements.txt 
```

## Usage

Start a game with main menu.

```bash
python ./main.py
```
